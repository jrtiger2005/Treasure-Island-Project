# 🏝️ Treasure Island - Python Adventure Game

Welcome to **Treasure Island**, a simple text-based adventure game built with **Python**. Your mission is to make the correct choices and find the hidden treasure. Every decision affects the outcome of the game!

---

## 📖 About the Project

Treasure Island is a beginner-friendly Python project that demonstrates:

- Conditional statements (`if`, `elif`, `else`)
- Nested conditions
- User input handling
- String manipulation
- Basic game logic
- Decision making

This project is perfect for anyone learning Python fundamentals.

---

## 🎮 Gameplay

The player goes through a series of choices such as:

- Left or Right
- Wait or Swim
- Choosing the correct door

Only one path leads to the hidden treasure, while the others result in Game Over.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Treasure-Island.git
```

2. Navigate to the project folder

```bash
cd Treasure-Island
```

3. Run the game

```bash
python treasure_island.py
```

or

```bash
python3 treasure_island.py
```
